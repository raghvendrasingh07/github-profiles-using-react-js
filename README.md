### Installation

To run this project, install it locally using npm:

```sh
$ git clone https://github.com/raghvendrasingh07/github-profiles-using-react-js.git
$ cd github-profiles-using-reactjs
$ npm install
$ npm start
```

For production environments...

```sh
$ npm run build
```
